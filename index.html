System Solver
The goal of the game is to find the value of an item based on the information you see on two (or three) shelves.

Here's an example where you would be asked to find the value of one car.

32

50

Start by noticing that there are four pumpkins on the top shelf.

This means that each pumpkin must have a value of 8.

There are six pumpkins on the bottom shelf, which means that the combined value of these pumpkins is 48.

Therefore, the remaining value on the bottom shelf is 2, and since there are two cars, each car must have a value of 1.
